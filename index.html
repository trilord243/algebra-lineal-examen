Solución del Parcial de Álgebra Lineal

Pregunta 1

Dada la transformación lineal T: ℝ³ → ℝ³ definida como:

T([x, y, z]) = [2x - 5y + 9z, -3x + 2y - 8z, 4x - 3y + 11z]

a) Hallar la matriz de transformación A_T en la base estándar de ℝ³ (2 puntos)

Para encontrar la matriz de transformación, aplicamos T a cada vector de la base estándar:

Base estándar de ℝ³:
e₁ = [1, 0, 0], e₂ = [0, 1, 0], e₃ = [0, 0, 1]
Aplicando T:
• T(e₁) = T([1, 0, 0]) = [2, -3, 4]
• T(e₂) = T([0, 1, 0]) = [-5, 2, -3]
• T(e₃) = T([0, 0, 1]) = [9, -8, 11]
Matriz de transformación:
A_T =
[ 2 -5 9]
[-3 2 -8]
[ 4 -3 11]

b) Hallar el espacio fila, espacio columna y núcleo con sus dimensiones (3 puntos)

Espacio fila:
Aplicando operaciones elementales por filas obtenemos la forma escalonada:
[ 2 -5 9]
[ 0 -5.5 5.5]
[ 0 0 0]
Resultados:
• Espacio fila: Span{(2, -5, 9), (0, -1, 1)}
• Dimensión del espacio fila: 2

• Espacio columna: Span{[2, -3, 4], [-5, 2, -3]}
• Dimensión del espacio columna: 2

• Núcleo: Span{[-2, 1, 1]}
• Dimensión del núcleo: 1

Pregunta 2

Dado el conjunto de vectores {v₁, v₂, v₃} = {(1, 0, 1), (-1, 1, -1), (2, 4, 0)}

a) Verificar si el conjunto es una base de ℝ³ (1 punto)

Matriz con vectores como columnas:
A =
[ 1 -1 2]
[ 0 1 4]
[ 1 -1 0]
Cálculo del determinante:
det(A) = 1×1×0 + (-1)×4×1 + 2×0×(-1) - 2×1×1 - 1×4×(-1) - 0×0×(-1)
det(A) = 0 - 4 + 0 - 2 + 4 + 0 = -2
Como det(A) = -2 ≠ 0, los vectores son linealmente independientes y forman una base de ℝ³.

b) Fórmulas del proceso de Gram-Schmidt (2 puntos)

Proceso de ortogonalización de Gram-Schmidt:

1. u₁ = v₁

2. u₂ = v₂ - proj_u₁(v₂)
   donde proj_u₁(v₂) = [⟨v₂,u₁⟩/⟨u₁,u₁⟩] × u₁

3. u₃ = v₃ - proj_u₁(v₃) - proj_u₂(v₃)

4. Normalización:
   e₁ = u₁/||u₁||, e₂ = u₂/||u₂||, e₃ = u₃/||u₃||

c) Calcular el conjunto de vectores ortonormales (3 puntos)

Paso 1: u₁ = v₁ = (1, 0, 1)

Paso 2: Calculamos u₂
• ⟨v₂, u₁⟩ = (-1)×1 + 1×0 + (-1)×1 = -2
• ⟨u₁, u₁⟩ = 1² + 0² + 1² = 2
• proj_u₁(v₂) = (-2/2)(1, 0, 1) = (-1, 0, -1)
• u₂ = (-1, 1, -1) - (-1, 0, -1) = (0, 1, 0)

Paso 3: Calculamos u₃
• ⟨v₃, u₁⟩ = 2×1 + 4×0 + 0×1 = 2
• ⟨v₃, u₂⟩ = 2×0 + 4×1 + 0×0 = 4
• proj_u₁(v₃) = (2/2)(1, 0, 1) = (1, 0, 1)
• proj_u₂(v₃) = (4/1)(0, 1, 0) = (0, 4, 0)
• u₃ = (2, 4, 0) - (1, 0, 1) - (0, 4, 0) = (1, 0, -1)
Paso 4: Normalización
• ||u₁|| = √2 → e₁ = (1/√2, 0, 1/√2)
• ||u₂|| = 1 → e₂ = (0, 1, 0)
• ||u₃|| = √2 → e₃ = (1/√2, 0, -1/√2)
Conjunto ortonormal:
{(1/√2, 0, 1/√2), (0, 1, 0), (1/√2, 0, -1/√2)}

Pregunta 3

Sabiendo que T(1,1) = (0,5) y T(-1,2) = (-3,1), encontrar T: ℝ² → ℝ² (6 puntos)

Buscamos una matriz de la forma:
[a b][x] = T([x])
[c d][y] [y]
De T(1,1) = (0,5):
• a + b = 0
• c + d = 5

De T(-1,2) = (-3,1):
• -a + 2b = -3
• -c + 2d = 1
Resolviendo el sistema:
• De a + b = 0: a = -b
• Sustituyendo en -a + 2b = -3: -(-b) + 2b = -3 → 3b = -3 → b = -1
• Por lo tanto: a = 1

• De c + d = 5: c = 5 - d
• Sustituyendo en -c + 2d = 1: -(5-d) + 2d = 1 → 3d = 6 → d = 2
• Por lo tanto: c = 3
Matriz de transformación:
A_T = [1 -1]
      [3 2]

Transformación lineal:
T([x, y]) = [x - y, 3x + 2y]

ii) Independencia lineal vs. Ortogonalidad (1 punto)

Afirmación: "Si dos vectores u y v son linealmente independientes, entonces son ortogonales."

Análisis:
• Dos vectores son ortogonales si ⟨u, v⟩ = 0
• Dos vectores son linealmente independientes si ninguno es múltiplo del otro

Contraejemplo:
• u = (1,0) y v = (1,1)
• Son linealmente independientes (ninguno es múltiplo del otro)
• Su producto escalar: ⟨u, v⟩ = 1×1 + 0×1 = 1 ≠ 0
• Por lo tanto, NO son ortogonales
Conclusión: La afirmación es FALSA. La independencia lineal no implica ortogonalidad.
Respuesta correcta: b) Falso
